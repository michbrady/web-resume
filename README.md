3. **Mobile testing**: Resize browser window or view on mobile device
4. **Dark mode**: Click the moon/sun icon in the top-right corner

## Customization

### Colors
The color scheme uses CSS custom properties and can be easily modified in `styles.css`:
- Primary: `#667eea` (Blue gradient start)
- Secondary: `#764ba2` (Purple gradient end)
- Dark: `#2c3e50` (Professional dark blue)

### Content
All content is in `index.html` and can be updated directly in the HTML structure.

### Animations
JavaScript animations can be customized in `script.js`. Current animations include:
- Fade-in on scroll
- Typing effect
- Hover transformations
- Parallax scrolling

## Deployment

### GitHub Pages
1. Create a new GitHub repository
2. Upload all files to the repository
3. Enable GitHub Pages in repository settings
4. Your resume will be available at `https://username.github.io/repository-name`

### Netlify
1. Drag and drop the folder to Netlify
2. Your resume will be live instantly with a custom URL

### Custom Domain
Point your domain to the hosting service for a professional URL like `resume.bradysbit.com`
